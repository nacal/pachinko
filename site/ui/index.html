UI Components

Live preview of @pachinko/ui Canvas drawing primitives.

7-Segment Display

Data lamp style digit rendering. Supports digits 0-9 with dim background segments and right-aligned multi-digit numbers.

import { drawSegmentDigit, drawSegmentNumber } from "@pachinko/ui";

// Single digits
drawSegmentDigit(ctx, "7", 10, 10, 20, 30, "#00ff66");

// Right-aligned number with dim background
drawSegmentNumber(ctx, 1234, x, y, 16, 24, "#ff4444", 4);

// Different colors for different stats
drawSegmentNumber(ctx, 42, x, y, 16, 24, "#00ff66", 4);   // green
drawSegmentNumber(ctx, 7, x, y, 16, 24, "#ff4444", 4);    // red
drawSegmentNumber(ctx, 319, x, y, 16, 24, "#ffaa00", 4);  // orange

Background, Grid & Axes

Foundation layers for chart rendering — background fill, grid lines, L-shaped axes, dashed zero line, and text labels.

import {
  resolveChartStyle, drawBackground,
  drawGridLines, drawAxes,
  drawZeroLine, drawAxisLabel,
} from "@pachinko/ui";

const style = resolveChartStyle();

drawBackground(ctx, w, h, style);
drawGridLines(ctx, left, top, cw, ch, 5, 4, style);
drawAxes(ctx, left, top, cw, ch, style);
drawZeroLine(ctx, left, zeroY, cw, style);
drawAxisLabel(ctx, "0", left - 6, zeroY, style,
  "right", "middle");

Bar

Filled rectangles with optional same-color border. Used for hit history charts.

import { drawBar } from "@pachinko/ui";

// Default border (0.6 opacity)
drawBar(ctx, x, y, 40, height, "#ff4444");
drawBar(ctx, x, y, 40, height, "#ffaa00");

// No border
drawBar(ctx, x, y, 40, height, "#00bfff", 0);

Circle & Empty Slot

Reserve ball indicators with solid colors, custom renderers, and empty slot placeholders.

import { drawCircle, drawEmptySlot } from "@pachinko/ui";

// Solid colors
drawCircle(ctx, x, y, 16, "#ffffff");
drawCircle(ctx, x, y, 16, "#4488ff");
drawCircle(ctx, x, y, 16, "#ff4444");
drawCircle(ctx, x, y, 16, "#ffd700");

// Custom renderer (gradient)
drawCircle(ctx, x, y, 16, (ctx, x, y, r) => {
  /* custom draw logic */
}, time);

// Empty slot
drawEmptySlot(ctx, x, y, 16);

No Data

Centered placeholder message for empty chart states.

import { drawNoData, resolveChartStyle } from "@pachinko/ui";

const style = resolveChartStyle();
drawNoData(ctx, width, height, style);

Combined: Stats Panel

A data lamp display built from @pachinko/ui primitives — 7-segment counters, labels, and hit history blocks. This is what @pachinko/tracker uses internally.

// Combine primitives to build complex displays
drawBackground(ctx, w, h, style);

// Column headers
ctx.fillStyle = "#88ff88";
ctx.fillText("回転数", col1X, y);

// 7-segment counters per column
drawSegmentNumber(ctx, 247, col1X, y,
  16, 30, "#00ff66", 4);
drawSegmentNumber(ctx, 3, col2X, y,
  16, 30, "#ff4444", 4);
drawSegmentNumber(ctx, 82, col3X, y,
  16, 30, "#ffaa00", 4);
