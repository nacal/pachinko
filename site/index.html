pachinko

A modular TypeScript engine for building browser-based pachinko experiences. Zero dependencies, fully deterministic, ready for the web.

Try the Demo API Reference

Packages

@pachinko/lottery

Weighted random draws, multi-stage lottery pipeline, game state transitions, and simulation. The mathematical core of the pachinko engine.

@pachinko/rendering

Canvas 2D reel animations with OffscreenCanvas + Worker support. Smooth spin/stop sequences with reach presentation effects.

@pachinko/effects

Declarative presentation effects — flash, shake, text overlays, and composable pipelines. Rule-based triggering tied to game events.

@pachinko/tracker

Session data tracking and Canvas chart visualization — slump graphs, hit history bar charts, and data lamp stats panels with customizable styles.

@pachinko/ui

Canvas-based UI component library — 7-segment digit displays, chart primitives, circle indicators, and shared theming utilities.

Features

Zero Dependencies

All packages have zero external runtime dependencies. Only browser APIs and pure TypeScript.

Deterministic RNG

Xoshiro128** PRNG with seed support. Reproducible results for testing and simulation.

OffscreenCanvas

Rendering runs in a Web Worker via OffscreenCanvas, keeping the main thread free. Automatic fallback for older browsers.

Fully Typed

Written in TypeScript strict mode with readonly types throughout. ESM + CJS dual exports.

Quick Start

npm install @pachinko/lottery @pachinko/rendering @pachinko/effects @pachinko/tracker
import { defineMachine, prob, createRng, draw, createState } from "@pachinko/lottery";

// Define a machine spec
const machine = defineMachine({
  name: "Demo Machine",
  modes: {
    normal:  { probability: prob(1, 319.68) },
    kakuhen: { probability: prob(1, 1.5), spins: 10 },
    jitan:   { probability: prob(1, 319.68), spins: 100 },
  },
  bonusTypes: [
    { id: "16R-kakuhen", label: "16R Kakuhen",
      category: "oatari", rounds: 16,
      nextMode: "kakuhen", nextModeSpins: null },
    { id: "4R-jitan", label: "4R Jitan",
      category: "koatari", rounds: 4,
      nextMode: "jitan", nextModeSpins: 100 },
  ],
});

// Draw!
const rng = createRng({ value: 42 });
const state = createState();
const result = draw(machine, rng, state);
console.log(result.outcome, result.reels);
